# Fatigue Prediction System 

##  Overview

This project predicts human fatigue levels (Low, Medium, High) using a machine learning model (Random Forest).

It also provides actionable recommendations to support decision-making in work environments.

---

##  Features

* Fatigue level prediction (Low / Medium / High)
* Work recommendation system:

  * Safe to work
  * Work with caution
  * Rest recommended
* Data-driven insights for improving human performance and safety

---

## ⚙️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Random Forest Classifier

---

## 📊 Dataset

The dataset includes features such as:

* Hours Awake
* Task Switches
* Sleep Hours
* Stress Level
* Cognitive Load

## 🏭 Practical Applications

### 🔹 Operational Level (Short-Term Decisions)

* The system can assist in real-time workforce management.
* Based on predicted fatigue levels, it can support decisions such as:

  * Job rotation
  * Adjusting task switching frequency
  * Deciding whether a worker should continue, proceed with caution, or take rest
* This helps in reducing immediate cognitive overload, minimizing errors, and improving safety.

---

### 🔹 Design Level (Long-Term Decisions)

* The system can be used to analyze fatigue trends over time across multiple workers.
* These insights can help organizations make strategic decisions such as:

  * Redesigning workflows to reduce unnecessary task switching
  * Introducing automation or new technologies to reduce human cognitive load
  * Improving shift planning and work distribution
* This enables the development of more efficient and human-centric systems.

---

### 🔹 Key Insight

This project demonstrates how machine learning can support both:

* **Operational control (short-term decision-making)**
* **System design improvements (long-term optimization)**
